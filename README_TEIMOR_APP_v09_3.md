# App TEIMOR · V09.3 Rendiment + Agenda

Versió neta dins carpeta pròpia.

## Canvis principals

- Pestanya Rendiment refeta per treballar sobre pressupostos/obres, amb files clicables per obrir directament el pressupost.
- Pressupostos i Rendiment agrupables per any actiu, amb selector d'any i opció “Tots”.
- Llibreria de partides agrupada per capítols amb opció d'editar el nom del capítol globalment.
- Línies del pressupost amb columna Capítol editable, a més de codi, unitat, concepte, quantitat i preu/ut.
- Nova pestanya Agenda tipus calendari mensual, amb cites/notes per dia, client, pressupost/obra activa o creació d'obra nova.
- Menú lateral més net: Còpies / JSON queda com una pestanya única.

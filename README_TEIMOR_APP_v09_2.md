# App TEIMOR Gestor Pressupostos · V09.2

Versió neta dins carpeta pròpia. Canvis principals:

- Llibreria de partides agrupada per capítols/grups.
- Ordenació amb fletxa ascendent/descendent a taules principals, especialment Pressupostos.
- Pressupostos ordenables per any, data, número, client, concepte, estat, base, total, tipus i nombre de partides.
- Modal de nou pressupost i edició de pressupost més gran.
- Camp de concepte/partida més ample i editable amb més espai.
- Número de pressupost automàtic en crear un nou pressupost, seguint el següent número disponible.
- Selector de partides de llibreria agrupat per capítols.

# TEIMOR · Gestor de pressupostos V09.6

Versió neta derivada de V09.5.

## Canvis V09.6

- Depurador de llibreria més fort per conceptes similars i no només per capítol tècnic.
- Detecció més estricta de textos que no són partides: IVA, totals, base imposable, fórmules, dates, dades de client i adreces.
- Mode de depuració "Molt fort" per reduir una llibreria importada massiva cap a una base més útil de partides tipus.
- Edició directa del capítol assignat dins la fitxa de cada partida.
- Camp de descripció llarga més llegible, amb lletra més gran.
- Previsualització A4 del pressupost amb format més modern i una mica de color corporatiu.
